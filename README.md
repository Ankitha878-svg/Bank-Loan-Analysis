# Bank-Loan-Analysis | SQL + Power BI

## 📌 Project Overview
This project presents an end-to-end Business Intelligence solution developed using SQL and Power BI to analyze bank loan data and evaluate loan portfolio performance.

SQL was used for data extraction, KPI calculations, and analytical queries, while Power BI was utilized for data transformation, modeling, DAX calculations, and creating interactive dashboards.

The dashboard provides insights into 38.6K loan applications with a total funded amount of $435.8M and total repayment amount of $473.1M.

---

## 🎯 Problem Statement
Financial institutions process thousands of loan applications, making it challenging to monitor loan performance, identify risky loans, and understand borrower trends.

This project aims to provide a centralized dashboard that helps stakeholders:
- Monitor overall loan portfolio performance.
- Analyze good and bad loan distribution.
- Track monthly lending and repayment trends.
- Identify customer borrowing patterns.
- Evaluate loan risk using borrower and loan attributes.

## 🔍 SQL Analysis Performed
SQL queries were used to analyze:
- Total and MTD loan applications.
- Total funded amount and amount received.
- Average interest rate and Debt-to-Income (DTI) ratio.
- Good vs Bad loan percentage and performance.
- Loan status analysis.
- Monthly loan trends.
- State-wise loan distribution.
- Loan term analysis.
- Employment length analysis.
- Loan purpose and home ownership analysis.

---

## 📈 Dashboard Features

### 1. Summary Dashboard
- Overall loan KPIs.
- Good Loan vs Bad Loan analysis.
- Loan status breakdown with funded amount, repayment amount, interest rate, and DTI.

### 2. Overview Dashboard
- Monthly loan application trends.
- Geographic loan distribution by state.
- Analysis by loan term.
- Employee length analysis.
- Loan purpose and home ownership insights.

### 3. Details Dashboard
Provides transaction-level loan information including:
- Loan ID.
- Loan purpose.
- Home ownership.
- Grade and sub-grade.
- Issue date.
- Funded amount.
- Interest rate.
- Installment amount.
- Amount received.

---

## 📊 Key Insights
- 86.2% of loans were categorized as good loans, indicating strong portfolio health.
- Debt consolidation represented the highest loan application category.
- The majority of customers opted for 36-month loan terms.
- Loan applications showed a consistent growth trend throughout the year.
- Home ownership and employment history influenced borrowing patterns.
